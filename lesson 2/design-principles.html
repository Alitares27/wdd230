Visual Hierarchy

BMW

https://www.bmw.com.pe/es/index.html

The visual hierarchy is important because we can determine what the user will read or view first. It allows us to order the elements of an interface according to their importance, to facilitate user interaction, it also allows us to facilitate the understanding of the information and guide the interaction towards the actions that are sought to be encouraged, by configuring elements such as: size, color, alignment, repetition, etc.

Hick's Law

TRAVEL1TOURS

https://travel1tours.com/

Thanks to Hick's law, we can calculate the user's response time when choosing an option among a number of stimuli. Since the response time in the choice of an option increases with a greater availability of different choices that are available. For example: The more options we have, the longer it takes us to decide and respond.

Fitt's Law

INTERBANK

https://interbank.pe/

This is a law that is basic to any interface design. Above all, on mobile devices where the CTA area must be large enough to avoid possible user errors. According to this law, faster movements towards smaller targets give a higher error rate. It is difficult to be accurate at high speed and with a small target. We must bear in mind that the time to acquire a target is a function of the distance and the size of the target.
